The R Bootcamp für CorrelAid
Juni 2021




1 - Installiere Base-R und RStudio auf deinem Laptop.

Verwende die folgenden Links um die aktuellen Versionen von R und RStudio zu installieren.

Installiere Base-R: R 4.1.0 (Windows), R 4.1.0 (Mac OS 10), R 4.1.0 (Mac OS 11)
Install RStudio: RStudio 1.4 (Windows), RStudio 1.4 (Mac)

2 - Installiere zusätzliche Pakete

In diese, Workshop werden wir verschieden R Pakete verwenden, die installiert sein müssen. Bitte führe den folgenden Code in deiner R Console in RStudio aus:

source('https://tinyurl.com/yc7vngfo')

Wenn Du gefragt wirst, ob du Pakete from source installieren möchtest, gebe n ein und drücke Enter.

3 - Optional: Zusätzliche Software installieren

Falls du Pakete kompilieren (from source) möchtest, oder das Installationsskript aus 2) Fehlermeldungen anzeigt, installiere bitte die folgende Software:

Windows: Folge der Anleitung auf der velinkten Seite
Mac: Command line tools (kein direkter Link verfügbar)
